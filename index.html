# Introduction | 소개

D.E.M. 은 Deadline Extension Machine, 제출기한 연장기입니다.
오늘도 과거의 게으름을 후회하며 과제를 제출할 위기에 놓인 당신을 위해 만들어졌죠 >_0

아, 이 머신은 원래 과제 제출기한 연장 프로그램에서 출발했어요.
해당 프로그램을 보고, 공부 반 재미 반으로 만들어졌답니다.

# Usage | 사용법
  • 제출할 파일 이름을 확장자를 포함해 입력하세요. (예: 보고서.hwp, 마지막_진짜_라스트.pptx)
  • 제출할 파일의 용량을 정해주세요.
  • 다운로드하세요.
    • 다운받은 파일의 용량과, 확장자가 의도한 대로 구성되었는지 확인하세요.
    • 정상적으로 읽히지 않는지도 꼭 확인하세요.
  • 제출하세요!
    • 마치 정말 과제를 힘들게 다 끝낸 것 처럼 메일 작성하는 것 잊지 마시구요.

자, 이제 제출기한이 연장되었습니다. 적어도 파일이 깨졌다는 연락이 올 때 까지는요.


파일의 용량이 클수록, 다운로드까지 오래 걸립니다.
